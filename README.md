# FlowTime

FlowTime - Staff management SaaS for The Flow beach bar

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
- ESLint + Prettier

## Getting Started
1. Clone the repo
2. Install dependencies: `npm install`
3. Copy `.env.local.example` to `.env.local` and fill in the Supabase credentials
4. Run the development server: `npm run dev`
